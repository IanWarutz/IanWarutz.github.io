Inspiration to start coding

"Any fool can write a code that a computer can understand. Good progrmammers write code that humans can understand".

Martin Fowler

Like any other skill, coding takes time and effort for you to master the information. The good thing is that you can always refer to your notes when developing your programs. There are many reasons why should consider coding as one of your skill sets:

  1. It is a well researched skill, taught by many people
  2. It enables you to be up to date with the current technological advancements
  3. It is highly marketable, thus employement is always a guarantee
  4. It enhances creativity
  5. It is a fun and interesting course

In any journey, if you ever want to give up, remind yourself the reason that you started. Also, find people who are interested in going along the same path as you. They will serve as a motivation to keep going even if you do not want to.

"Perfection is achieved not when there is nothing more to add, but rather when there is nothing more to take away".

Antoine de Saint-Exupery

I wish you all the best. May you complete your journey and become a successful web developer.
